1 Hitelviszonyt megtestesítő értékpapírok típusai, árazása.

  pénzügyi piacok csoportosítása
    időbeliség alapján
      - pénzpiac (1 évnél rövidebb futamidő)
      - tőkepiac (1 évnél hosszabb futamidő)
    kereskedett eszközök alapján
      - hitelpiac
      - értékpapírpiac (elsődleges / másodlagos piac)
      - devizapiac
      - derivatív piac
    szervezettség alapján
      - tőzsde
      - OTC

  értékpapírok fogalma
    külső feltételtől nem függő, egyoldalú kötelezettségvállalás
    megfelelő jogszabályi kellékekkel rendelkező okirat
    közgazdaságtan
      vagyonnal kapcsolatos, jogot megtestesítő forgalomképes okirat
      jelenbeli pénzek jövőbeni pénzre történő cseréje
    jog
      minden okirat, adat, aminek kiállítását, megjelenését
      jogszabály lehetővé teszi
    Polgári Törvénykönyv (2013. évi V.) Hatodik Könyv Ötödik Rész
      egyoldalú jognyilatkozat
      papíralapú vagy más módon nyilvántartott (dematerializált értékpapírként)
      benne foglalt jogot gyakorolni csak az értékpapír által lehet

  értékpapírok osztályozása
    alapjogviszony alapján
      - pénzkövetelés: pl. kötvény
      - tagsági viszony: pl. részvény
      - dologra vonatkozó tulajdonjog vagy más jog: pl.közraktárjegy
    átruházhatóság alapján
      - bemutatóra szóló: pl. forgatható csekk
      - névre szóló: pl. nyomdai úton kibocsátott kötvény
      - rendeletre szóló: pl. váltó
    megjelenése alapján
      - egyedi: pl. csekk
      - sorozat: pl. ötvény, részvény
    lejárat alapján
      - rövid lejáratú:  pl. kincstárjegy, csekk
      - középlejáratú:   pl. kötvény, letéti jegy
      - hosszú lejáratú: pl. kötvény, jelzáloglevél
      - lejárat nélküli: pl. részvény

    egyéb kategóriák
      - kamatozás
      - kibocsátó (állam, vállalat - hitelintézet, magánszemély)
      - tőzsdeképesség
      - kibocsátás módja (nyilvános, zártkörű)
      - egyéb (pl. elévülés)

  váltó
    eredetileg: pénzhelyettesítő, hiteleszköz
    1930: genfi váltójogszabály (Magyarország: 1965)
    legpontosabban kidolgozott értékpapír
    forgatása forgatmány útján (megtérítési váltóadósok)
    fajtái
    - idegen váltó (fizetési felszólítás) pl. exportőr által kiállított
      kibocsátó, rendelvényes (kedvezményezett), címzett
    - saját váltó (fizetési ígéret) pl. importőr által kiállított

  csekk
    pénzkövetelés, nem hitelviszony
    készpénz helyettesítő fizetési eszköz
    kibocsátó aláírásával, általa meghatározott tartalommal jön létre
    címzett: bank (ahol a kibocsátónak követelése van)
    megtekintéskor fizetendő

  közraktárjegy
    közraktárban elhelyezett áruk helyettesítésére szolgál
    egyedi értékpapír
    rendeletre szól
    alapjogviszonya árura vonatkozó tulajdonjog vagy más jog
    kibocsátó: közraktár, zálogjog illeti meg
    letevő kapja az áru átvételének elismerését
    két rész
    - árujegy (dolog feletti rendelkezési jog)
    - zálogjegy (hitelfelvételnél zálogként használható)
      mindkettőt vissza kell adni, hogy visszakapjuk az árut

  kötvény
    hitelviszony: kibocsátó visszafizeti, teljesíti
      pénzösszeg + kamatok + járulékok + egyéb szolgáltatások
    tőzsdei
    kamatozás: fix, lebegő, kamatos kamatozású, nem kamatozó
    futamidő: általában közép vagy hosszú
    kibocsátás: jogi személyiséggel rendelkező gazdálkodó szervezetek
    speciális típusai
      átváltható (átváltoztatható, átváltozó)
      opciós: jegyzési jog biztosít

    állampapírok
      hitelviszony, állammal szembeni pénzkövetelés (nem évül el)
        - kincstárjegy (éves költségvetési törvény)
          jellemzően éven belüli lejáratú állampapírok
        - kötvény
      kárpótlási jegy
      - állammal szembeni követelést névértékben megtestesítő értékpapír

  részvény
    részvénytársaság által kibocsátott tagsági jogot megtestesítő értékpapír
    kibocsátást megelőzően: részvényutalvány, ideiglenes részvény
    részvényes felelőssége: részvény névértékére terjed ki
    - alaptőke egy részét képviseli
    - forgalomra szánt

    jogok
    - tagsági (közgyűlés, szavazati jog, felvilágosítás, kisebbségi jogok)
    - vagyoni (osztalék, likvidációs hányad)

    típusok
    - törzsrészvény
    - elsőbbségi
    - kamatozó
    - dolgozói
    - visszaváltható

  jelzáloglevél
    hosszú lejáratú, átruházható, általában sorozatban kibocsátott
    jelzálog hitelintézet bocsátja ki
    jelzáloghitel: mögötte bejegyzett zálogjog áll biztosítékként
    MBS = mortgage backed security = jelzálogfedezetű értékpapír
      NEM jelzálog levél, ennek jelzáloghitel pool a fedezete

  pénzpiaci eszközök
    rövid lejáratú eszközök a nemzetközi pénzpiacokon
    - kincstárjegyek
    - CD = certificate of deposit
      letéti jegy rövid lejáratú, forrásgyűjtésre szolgáló értékpapír
      főleg wholesale banking termék (pénzügyi intézményeknek bocsátják ki)
    - CP = commercial paper = kereskedelmi papír
      rövid lejáratú értékpapír
      fontos a bankközi piacon (bankok likviditásához)
    - Eurojegyek
      rövid lejáratú, hazai valutától eltérő pénznemben kibocsátott papír
      forgatással átruházható
      különféle formák

  kötvények árazása

    jelenérték - jövőérték
      FV = PV * (1 + r/m)^(n*m)
      r: éves kamatláb
      m: kamatfizetések száma egy év alatt
      n: futamidő években
      PV: jelenérték
      FV: jövőérték

    nettó jelenérték: jövőbeli pénzáramok jelenértéke
      NPV = sum_i C_i / (1 + r_i)^i
    effektív kamatláb: r_eff = (1 + r/m)^m - 1

    annuitás: n tagú, fix összegű pénzáramlás
      PV = C * (1/r - 1/(1+r)^n)  mértani sorozat összege
      C: pénzáramlás összege

    örökjáradék: végtelen tagú, fix összegű pénzáramlás
      PV = C / r

    növekvő tagú örökjáradék
      PV = C1 / (r - g)  ha r > g
      g: növekedési ütem

    kötvények árfolyama
      nettó jelenérték alapján
      P_0 = kamatok + névérték = sum_t C_t / (1 + r)^t + P_n / (1 + r)^n
      amerikai kötvények: féléves kamatfizetési periódus
      amortizálódó kötvények: egyenletes tőketörlesztés is történik
      bruttó árfolyam = nettó árfolyam + felhalmozódott kamat

    névérték vs árfolyam
      - diszkont áron kibocsátott kötvény
        névérték > árfolyam -> piaci kamat > névleges kamat
      - prémium kötvény
        névérték < árfolyam -> piaci kamat < névleges kamat

  kötvények hozama

    CY  = coupon yield = szelvényhozam
      névleges kamatláb / nettó piaci árfolyam
      nettó árfolyam = bruttó árfolyam - felhalmozott kamat
    SYTM = simplified yield to maturity = lejáratig számított egyszerű hozam
      szelvényhozam + lejáratig számított évesített árfolyamváltozás
      különbség a CY-tól: kötvény névértéken kerül visszaváltásra,
        de nem névértéken kerül megvásárlásra
    YTM = yield to maturity = lejáratig számított hozam
      tényleges hozam: kötvény belső megtérülési rátája
      az a hozam, aminél a jelenérték 0
      pontosabb, mint az SYTM
    HPR = holding period return = befektetési időtávra jutó hozam
      teljes hozam a befektetés idejére
      HPR = osztalék / ár + árfolyamváltozás / ár
      nincs évesítve
