2 Elméleti eloszlások

  valószínűségi változó fogalma, eloszlásfüggvény és sűrűségfüggvény

    valószínűségi változó
      a valószínűségi változó azzal jellemzi a véletlen jelenséget,
      hogy értékeit csak bizonyos valószínűséggel veszi fel

    valószínűségi változó típusai
      diszkrét
      folytonos

    eloszlásfüggvény
      definíció: eloszlásfüggvény megadja, hogy egy xi valószínűségi változó
        mekkora valószínűséggel vesz fel egy adott x-nél kisebb értéket
        F(x) = P(xi < x)
      tulajdonságai
        balról folytonos
        monoton növekvő: ha a < b, akkor F(a) <= F(b)
        határértékei 0 és 1: F(-infty) = 0, f(infty) = 1

    sűrűségfüggvény
      definíció: ha xi folytonos valószínűségi változó F(x) eloszlásfüggvénnyel,
        akkor az f(x) sűrűségfüggvény az F(x) eloszlásfüggvény deriváltja
      tulajdonságai
        nem negatív: f(x) >= 0
        integrálja 1: int f(x) dx = 1
      várható érték
        int x f(x) dx

    eloszlásfüggvény és sűrűségfüggvény kapcsolata
      P(a <= X < b) = F(b) - F(a) = int_a^b f(x) dx

  diszkrét elméleti eloszlások (binomiális, Poisson) alkalmazásai, tulajdonságai

    binomiális eloszlás
      egy A bekövetkezésének valószínűsége: P(A) = p
      kísérlet n független ismétlése
      X = A esemény bekövetkezéseinek száma

      összefüggések
        - súlyfüggvény: p_k = P(X = k) = (n alatt k) * p^k * (1 - p)^(n-k)
        - várható érték: E(X)   = n * p
        - variancia:     D^2(X) = n * p * (1-p)

      alkalmazás
        esemény bekövetkezésének / nem bekövetkezésének vizsgálata
        minőségmenedzsment: visszatevéses mintavétel
        ha n nagy, módusz = várható érték

    Poisson eloszlás
      másik név: ritka események eloszlástörvénye
      véletlen pontelhelyezkedések vizsgálata
      véletlenszerűen elhelyezkedő pontok száma egy adott tartományban

      összefüggések
        - súlyfüggvény: p_k = P(X = k) = lambda^k / k! * e^(-lambda)
        - várható érték: E(X)   = lambda
        - variancia:     D^2(X) = lambda

      alkalmazás
        egyik leggyakoribb eloszlás
        ritka események: pl. gépelési hibák száma egy oldalon
        binomiális eloszlás közelítése, ha n nagy és p kicsi
          lambda = n*p
        módusz, várható érték közel azonos

  folytonos elméleti eloszlások (exponenciális, Gauss) alkalmazásai, tulajdonságai

    exponenciális eloszlás
      véletlen hosszúságú időtartamok eloszlása

      összefüggések
        - sűrűségfüggvény:  f(x)   = lambda * exp(- lambda * x)
        - eloszlásfüggvény: F(x)   = 1 - exp(- lambda * x)
        - várható érték:    E(X)   = 1 / lambda
        - variancia:        D^2(X) = 1 / lambda^2

      alkalmazás példák
        - berendezés élettartama / hibamentes működési ideje
          ha nem kopás, hanem véletlen hiba miatt következik be
        - radoaktív atom élettartama
        - telefonbeszélgetések időtartama

    normális eloszlás
      leggyakoribb eloszlás

      CHT = központi határeloszlás tétele
        sok független, hasonló nagyságrendű tényező
        tényezők hatása összeadódik -> határértékben normális eloszlást kapunk

      összefüggések
        - sűrűségfüggvény
          f(x) = 1 / (s * sqrt(2 pi)) * exp(- (x - mu)^2 / (2s^2))
        - eloszlásfüggvény
          F(x) = int_[- infty, x] f(x) dx
          nem adható meg zárt alakban
        - várható érték: E(X)   = mu
        - variancia:     D^2(X) = s

      standardizálás
        cél: standard normális eloszlássá konvertálás (jelölés: Phi(x))
        F(x) = Phi((x - mu) / s)
        standard normális eloszlás szimmetrikus a 0 körül

      háromszigma szabály
        +- 1 szórásra a várható értéktől: adatok 68.26%-a
        +- 2 szórásra a várható értéktől: adatok 95.44%-a
        +- 3 szórásra a várható értéktől: adatok 99.73%-a

      alkalmazás
        arányos skálán mérhető termékjellemzők (szélesség, hosszúság)
        technológiai paraméterek eloszlása (húmérséklet, nyomás)
        véletlen jellegű mérési hibák eloszlása
        szabályozási algoritmusok
